Run the `dealer-performance-intelligence` skill. This generates a comprehensive performance benchmarking report comparing your dealer group against the full ~400 dealer group industry cohort and named public peers (Carvana, CarMax, Sonic, Penske, etc.) on used vehicle DOM, YoY growth, DOM trend, new vehicle pricing, and inventory mix.

Optional argument: location name to scope to a single location. Without argument, runs for all locations.
